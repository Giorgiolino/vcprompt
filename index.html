VCPROMPT

version control info in your prompt. see the source.

INSTALL

$ curl -sL https://github.com/xvzf/vcprompt/raw/master/bin/vcprompt > ~/bin/vcprompt 
$ chmod +x ~/bin/vcprompt
$ export PS1="[$(vcprompt)] $PS1"

EXAMPLES

what systems are supported?

$ vcprompt --systems
bzr
darcs
fossil
git
hg
svn

the default output is '<system>:<branch>'

$ vcprompt
git:master

show only the hash instead of the branch

$ vcprompt -f '%s:%h'
git:bc3c3fb

-p/--path can be use to specifiy a path to run vcprompt

$ vcprompt --path ~/anotherproject
hg:default
